# **App Name**: Sloth Vision

## Core Features:

- Interactive Scratchcard: Implement a virtual scratchcard where users swipe/click to reveal a random prediction.
- Prediction Reveal: Display a randomly selected, fictional prediction upon scratching (e.g., 'Goal at 90!', 'Historic Comeback!').
- TikTok Video Carousel: Embed a carousel of 4 TikTok videos that loop automatically.
- External Link: Provide a neon green CTA button that directs users to an external page (PAGE 2).
- Analytics: Monitor interactions, focusing on engagement metrics for the scratchcard and clicks to the external page. Consider if you can send this data to Firestore.
- Mobile Optimization: Ensure 100% responsiveness and mobile-first design for TikTok integration.

## Style Guidelines:

- Primary color: Grass green (#1A5F1A) to evoke the feeling of a soccer field.
- Background color: Off-white (#F0F0F0) to provide a clean and readable backdrop.
- Accent color: Neon green (#39FF14) to highlight call-to-action buttons.
- Headline font: 'Bebas Neue' (sans-serif) for titles and prominent text. Note: currently only Google Fonts are supported.
- Body font: 'Inter' (sans-serif) for body text and descriptions. Note: currently only Google Fonts are supported.
- Use animated icons for trust-building cards, related to the description of each card.
- Mobile-first design with a fixed header and footer for easy navigation.
- Subtle animations for the scratchcard effect and loading transitions to improve user experience.
